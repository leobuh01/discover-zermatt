## About This Project

This is the **Discover Zermatt** tourism destination website, created as part of the Tourism Intermediation course at UIB (Universitat de les Illes Balears), Mallorca.

The website simulates a DMO (Destination Management Organization) website for Zermatt, Switzerland, covering all 5 course activities.

## How to View on GitHub Pages

1. Go to your GitHub repository
2. Click **Settings**
3. Scroll to **Pages**
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at: `https://[yourusername].github.io/[repository-name]/`
